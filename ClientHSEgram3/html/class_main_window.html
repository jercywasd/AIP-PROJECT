Подробное описание

Класс окна авторизации и регистрации пользователей.

  • Предоставляет графический интерфейс (GUI) для ввода учетных данных, выполняет клиентскую валидацию строк, осуществляет криптографическую защиту паролей с помощью Argon2id и управляет жизненным циклом сетевого сокета до момента входа в чат.

◆ sockReady

void MainWindow::sockReady ( )
slot

Слот для обработки входящих данных от сервера (readyRead).

  • Читает входящий поток байт, преобразует его в JSON-документ и маршрутизирует логику в зависимости от ответа сервера:
  • "type": "login" при успехе отключает сигналы, динамически создает ChatWindow, передает ему сокет, делает окно чата главным и скрывает текущее окно авторизации.
  • "type": "askhash" сверяет введенный пользователем пароль с полученным хэшем Argon2id. При совпадении формирует запрос на авторизацию и инициирует (или использует) подключение к серверу.
  • "type": "check_login" при успешной предварительной проверке логина хэширует пароль через Argon2id, отправляет финальный пакет регистрации "type": "reg" и переключает интерфейс на форму входа.
