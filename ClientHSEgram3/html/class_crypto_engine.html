Подробное описание

Класс-утилита для выполнения криптографических операций.

  • Обертка над библиотекой libsodium, предоставляющая методы для генерации ключей X25519, вычисления сессионных ключей (Shared Key) и симметричного шифрования/дешифрования сообщений по алгоритму ChaCha20-Poly1305.

◆ decryptMessage()

QString CryptoEngine::decryptMessage ( const QJsonObject & payload,
const QByteArray & sharedKey )
static

Расшифровывает сообщение, зашифрованное алгоритмом ChaCha20-Poly1305 (IETF).

  • Извлекает зашифрованный текст и nonce из JSON-объекта, декодирует их из Base64, проверяет целостность данных (имитовставку Poly1305) и подлинность сообщения. Если данные были изменены в процессе передачи, расшифровка завершится ошибкой.
  • Аргументы
    payloadJSON-объект с полями "ciphertext" и "nonce" в формате Base64.
    sharedKeyОбщий секретный сессионный ключ.
    Возвращает
    QString Восстановленный открытый текст (в UTF-8), либо сообщение об ошибке в случае нарушения целостности или невалидных ключей.

◆ encryptMessage()

QJsonObject CryptoEngine::encryptMessage ( const QString & message,
const QByteArray & sharedKey )
static

Шифрует текстовое сообщение с использованием алгоритма ChaCha20-Poly1305 (IETF).

  • Функция преобразует строку в UTF-8, генерирует случайный одноразовый код (nonce) с помощью криптографически стойкого генератора randombytes_buf и выполняет шифрование с аутентификацией данных (AEAD). Результат упаковывается в JSON.
  • Аргументы
    messageИсходный открытый текст сообщения.
    sharedKeyОбщий секретный сессионный ключ.
    Возвращает
    QJsonObject JSON-объект, содержащий зашифрованный текст ("ciphertext") и одноразовый код ("nonce"), закодированные в Base64. В случае ошибки возвращает пустой объект.

◆ FindSharedKey()

QByteArray CryptoEngine::FindSharedKey ( const QByteArray & myPrivateKey,
const QByteArray & theirPublicKey )
static

Вычисляет общий секретный ключ (Shared Key) на основе схемы Диффи-Хеллмана.

  • Функция crypto_box_beforenm выполняет скалярное умножение на эллиптической кривой X25519, используя собственный приватный ключ и публичный ключ собеседника. Полученный результат далее используется для симметричного шифрования сообщений.
  • Аргументы
    myPrivateKeyСвой приватный ключ (размером crypto_box_SECRETKEYBYTES).
    theirPublicKeyПубличный ключ собеседника (размером crypto_box_PUBLICKEYBYTES).
    Возвращает
    QByteArray Общий секретный ключ (размером crypto_box_BEFORENMBYTES), либо пустой массив в случае ошибки.

◆ GenerateKeyPair()

QPair< QByteArray, QByteArray > CryptoEngine::GenerateKeyPair ( )
static

Генерирует новую асимметричную пару ключей (публичный и приватный).

  • Использует функцию crypto_box_keypair для создания пары ключей, предназначенной для обмена ключами и шифрования (X25519).
  • Возвращает
    QPair<QByteArray, QByteArray> Пара, где первый элемент (first) — публичный ключ, а второй (second) — приватный (секретный) ключ.

◆ init()

bool CryptoEngine::init ( )
static

Инициализирует библиотеку libsodium.

  • Данную функцию необходимо вызвать один раз перед использованием любых других криптографических функций библиотеки. Безопасна для повторного вызова.
  • Возвращает
    true Если библиотека успешно инициализирована (или уже была инициализирована).
    false Если произошла критическая ошибка при инициализации.
