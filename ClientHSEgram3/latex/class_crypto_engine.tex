\doxysubsection{Подробное описание}
Класс-\/утилита для выполнения криптографических операций. 


\begin{DoxyItemize}
\item Обертка над библиотекой libsodium, предоставляющая методы для генерации ключей X25519, вычисления сессионных ключей (Shared Key) и симметричного шифрования/\+дешифрования сообщений по алгоритму Cha\+Cha20-\/\+Poly1305. 
\end{DoxyItemize}
