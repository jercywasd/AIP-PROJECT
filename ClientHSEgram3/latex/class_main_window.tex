\doxysubsection{Подробное описание}
Класс окна авторизации и регистрации пользователей. 


\begin{DoxyItemize}
\item Предоставляет графический интерфейс (GUI) для ввода учетных данных, выполняет клиентскую валидацию строк, осуществляет криптографическую защиту паролей с помощью Argon2id и управляет жизненным циклом сетевого сокета до момента входа в чат. 
\end{DoxyItemize}

Слот для обработки входящих данных от сервера (ready\+Read). 


\begin{DoxyItemize}
\item Читает входящий поток байт, преобразует его в JSON-\/документ и маршрутизирует логику в зависимости от ответа сервера:\+
\item {\ttfamily "{}type"{}:\+ "{}login"{}} при успехе отключает сигналы, динамически создает {\ttfamily \doxylink{class_chat_window}{Chat\+Window}}, передает ему сокет, делает окно чата главным и скрывает текущее окно авторизации.
\item {\ttfamily "{}type"{}:\+ "{}askhash"{}} сверяет введенный пользователем пароль с полученным хэшем Argon2id. При совпадении формирует запрос на авторизацию и инициирует (или использует) подключение к серверу.
\item {\ttfamily "{}type"{}:\+ "{}check\+\_\+login"{}} при успешной предварительной проверке логина хэширует пароль через Argon2id, отправляет финальный пакет регистрации {\ttfamily "{}type"{}:\+ "{}reg"{}} и переключает интерфейс на форму входа. 
\end{DoxyItemize}
