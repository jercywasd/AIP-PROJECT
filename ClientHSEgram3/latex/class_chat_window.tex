Слот для обработки входящих данных из сокета (ready\+Read). 


\begin{DoxyItemize}
\item Читает сырые данные от сервера, парсит их в JSON и распределяет обработку в зависимости от поля "{}type"{}:\+
\item "{}start"{}:\+ инициализация диалогов текущего пользователя.
\item "{}message\+To"{}:\+ обработка входящего сообщения (дешифрование и вывод на экран или инкремент счетчика непрочитанных).
\item "{}find\+\_\+dialog"{}:\+ обработка результата поиска/\+создания диалога.
\item "{}get\+\_\+public\+\_\+key"{}:\+ получение и сохранение публичного ключа собеседника.
\item "{}history\+\_\+result"{}:\+ загрузка, дешифрование и отображение истории сообщений. 
\end{DoxyItemize}
