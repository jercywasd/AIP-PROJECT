Подробное описание

Сетевой TCP-сервер мессенджера на базе QTcpServer.

  • Осуществляет координацию клиентов, централизованное хранение сообщений в SQLite, распределение публичных криптографических ключей участников для организации E2E-шифрования, а также проверку подлинности сессий.

void HseServer::startServer ( )

Запускает сетевой сервер и инициализирует базу данных.

  • Поднимает прослушивание (listening) на локальном адресе LocalHost и порту 5531. В случае успеха настраивает и открывает подключение к SQLite базе данных hseDB.db.
