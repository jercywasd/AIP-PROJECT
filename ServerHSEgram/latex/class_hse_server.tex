\doxysubsection{Подробное описание}
Сетевой TCP-\/сервер мессенджера на базе QTcp\+Server. 


\begin{DoxyItemize}
\item Осуществляет координацию клиентов, централизованное хранение сообщений в SQLite, распределение публичных криптографических ключей участников для организации E2\+E-\/шифрования, а также проверку подлинности сессий. 
\end{DoxyItemize}
