# ClientHSEgram

**ClientHSEgram** — это кроссплатформенный клиент безопасного мессенджера, разработанный на C++ с использованием фреймворка Qt6. Проект реализует полноценное сквозное шифрование (End-to-End Encryption) сообщений на базе современных криптографических протоколов библиотеки `libsodium`.

## 🚀 Основные возможности

* **Графический интерфейс (UI):** Интуитивное и современное рабочее пространство на базе Qt Widgets. Включает главное окно авторизации/поиска и отдельное окно чата (`ChatWindow`).
* **Безопасность (E2E Encryption):** * Генерация пар ключей (Curve25519).
  * Вычисление общего секрета (Shared Secret) методом Диффи-Хеллмана.
  * Симметричное шифрование каждого сообщения по алгоритму ChaCha20-Poly1305.
  * Защита от модификации данных и атак повторения при помощи уникальных одноразовых чисел (`nonce`).
* **Сетевое взаимодействие:** Обмен структурированными JSON-пакетами с сервером через защищенные сокеты.
* **Модульное тестирование:** Автоматическое покрытие критически важных узлов шифрования и сетевой логики через фреймворк `Qt Test` и утилиту `CTest`.

## 🛠 Архитектура проекта

Проект разделен на изолированные логические компоненты:
* `CryptoEngine` — статическое ядро, отвечающее за работу с `libsodium`, шифрование, дешифрование и генерацию ключей.
* `ChatWindow` — контроллер интерфейса окна чата, управляющий вводом, отображением истории и сериализацией пакетов.
* `MainWindow` — стартовое окно приложения (авторизация, списки контактов).

## 📋 Требования к окружению

Для сборки и запуска проекта вам понадобятся:
1. **Операционная система:** Windows 10/11, Linux или macOS.
2. **Компилятор:** GCC (MinGW для Windows) версии 11.0+ или Clang с поддержкой стандарта **C++17**.
3. **Система сборки:** CMake версии 3.16 или выше.
4. **Фреймворк:** Qt6 (модули `Core`, `Widgets`, `Network`, `Sql`, `Multimedia`, `Test`).
5. **Библиотека криптографии:** `libsodium` (скомпилированная под целевую архитектуру).
